# Sample pages for frontend developers

## Stack

- Next.js
- Tailwind CSS
- Ant Design (Design System)

## Atomic Design Pattern

- page.tsx
  - contains Container
- Container
  - contains Template
  - handle all props
  - handle business logics
- Template
  - contains Modules
  - set layout
- Module
  - contains Components and Atoms
- Component
- Atom
  - Basic component declared in Figma
